Для запуска команд потребуется любой диструбутив Linux, так же подайдёт Mac, но некоторые команды там работают иначе
В целом вся суть лаб в том, чтобы научить вас пользоваться гуглом и командой man
Все команды выполняются в терминале, для смены каталога использутся команда:
cd папка
чтоб узнать в какой ща папке подходит такая:
pwd
чтоб увидеть список папок в текущем каталоге пиши
ls -la
